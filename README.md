# Steps:

- Init the project

```sh
npm init -y
```

- Install typscript

```sh
npm install -D typescript
```

- Setup the typescript config

```sh
npx tsc --init
```

- Update `outDir` and `rootDir` in the `tsconfig.json` file

- Run the compilation command

```sh
npx tsc
```

- Run the typescript file using `tsx`

```sh
npm install -g tsx
```
